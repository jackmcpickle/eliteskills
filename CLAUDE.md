In all interaction and commit messages be extremely concise.

## Commands

```bash
# Development
pnpm dev                 # Start dev server on port 3000
pnpm build               # Production build (astro and wrangler )
pnpm preview             # Preview production build

# Testing & Quality
pnpm check               # Astro type checking (astro check)
pnpm lint                # Lint via oxlint - DO NOT npx tsc --notEmit
pnpm fmt                 # Prettier fix
```
